## 🧭 What this is

broker-as-a-service lets you get cross-chain crypto swap quotes, start swaps, and track progress in one place. It works with Chainflip and supports common chains like Bitcoin, Ethereum, and Solana.

This app is built for people who want a simple way to move crypto between chains without dealing with a long setup.

## 💻 What you need

- A Windows 10 or Windows 11 PC
- An internet connection
- A crypto wallet for the chain you plan to use
- Enough funds for the swap and network fees

## ▶️ Start using the app

1. Open broker-as-a-service from the Start menu or desktop shortcut
2. Add the wallet or account details needed for your swap
3. Choose the asset you want to send
4. Choose the asset you want to receive
5. Review the quote
6. Start the swap

The app will guide you through each step. You only need to follow the prompts on screen.

## 🔍 Get a quote

Use the quote screen to see the estimated swap amount before you send anything.

You can check:
- The asset you send
- The asset you receive
- The estimated rate
- The expected network fees
- The swap route

This helps you confirm the trade before you continue.

## 🔁 Track swap progress

After you start a swap, you can follow its status in the app.

Common status steps:
- Quote created
- Swap started
- Funds detected
- Swap processing
- Swap complete

If the swap takes time, keep the app open and check the progress view.

## 🧰 Supported swap types
